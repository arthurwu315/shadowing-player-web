# YouTube 跟讀播放器（GitHub Pages 版）

這是一個單檔 `index.html` 的 YouTube 跟讀工具，目標是「免伺服器、可直接放 GitHub Pages、手機可用」。

## 目前功能

- 多主題介面（Tailwind CSS + 本地主題切換）。
- 貼上 YouTube 網址後自動載入影片。
- 透過免金鑰字幕 API 嘗試抓字幕（先 Supadata，再備援來源）。
- API 失敗時會顯示手動流程提示：`自動抓取受限，請點擊下方按鈕手動貼入英/中雙語講稿以開始練習`。
- 手動貼上雙語字幕：支援「英文講稿」與「中文翻譯」兩欄同時貼上。
- 智慧格式解析與對齊：
  - 格式 A（帶時間）：`00:12 Hello world`
  - 格式 B（時間與文字分行）：如 `0:03` 下一行是句子內容。
  - 格式 C（純文字）：每行一句，依「每句秒數」自動分配時間軸。
  - 兩邊都有時間戳記時，會自動按時間配對。
  - 純文字時，會按行數對齊。
  - 單邊有時間、另一邊純文字時，會對齊到有時間的一側。
- 列出所有字幕行。
- 點擊任一字幕行可跳到該句開始時間。
- 每行都有 `🔁 循環` 按鈕，可單句重複播放。
- Auto-Pause 自動留白模式：句尾自動暫停 `該句時長 x 1.2`，再跳到下一句。
- 快捷大按鈕：`上一句`、`重新播放本句`、`下一句`。
- 播放速度快捷切換：`0.75x`、`1x`、`1.25x`。
- 計數器：每句可累積練習次數（點擊跳句 +1、句子播完 +1），並儲存到 `localStorage`。
- 盲聽模式：預設字幕模糊，滑鼠懸停或手指按住時才清晰。
- 句子級播放控制：每句字幕卡片新增 `⏯ 播放/暫停`，可就地中斷或續播練習。
- 單句播放鍵即時狀態：按鈕會依播放器狀態切換 `▶️ / ⏸️`。
- 字幕卡片極簡化：每句僅保留 3 個主動作（時間跳轉、播放、循環），降低視覺雜訊。
- 全局顯示控制面板：
  - 全部顯示英文
  - 全部顯示中文
  - 全部顯示雙語
  - 全部模糊（盲聽）
  - 僅顯示當前句子（Focus Mode）
- 全域懸浮控制塔：播放器與全局控制面板整合為頂部固定的毛玻璃控制區。
- 藥丸按鈕 UI：全局控制改為圖示 + 文字的 pill 風格，點擊辨識更直覺。
- Lo-Fi Pixel 視覺規範：暖米色底、2px 黑邊、硬陰影與莫蘭迪按鈕配色。
- 控制塔瘦身：手機上按鈕區以壓縮高度呈現，避免遮擋字幕主要閱讀區。
- 滑動自動收合：向上捲讀時自動收合控制面板僅留播放器，向下滑或暫停自動展開。
- 邏輯解耦：Focus Mode 與 Blind Mode 可獨立開啟，不互相觸發。
- 經典跟讀片單：內建 3 個影片卡片，點擊可自動填入影片 ID。
- 手機友善：按鈕高度加大、間距加大，方便點擊。
- 記憶優化：同影片 ID 會記住雙語內容、每句英中顯示狀態、全局顯示設定與計數器（`localStorage`）。
- 當前句自動顯現：即使在單句循環或全局模糊中，播放器當前句會自動清晰顯示，保持跟播放進度同步。
- 調速穩定性：切換 `0.75x / 1x / 1.25x` 不會讓當前句消失或重新被模糊。
- 自動置中捲動：進入下一句時，會平滑捲動把當前句置中，手機上更好跟讀。
- 高頻同步：時間偵測提升至每 100ms 一次，降低切句與圖示反應延遲。
- 當前句視覺焦點：Lo-Fi 像素綠/肉桂橙邊框 + 陰影 + 微放大，提升跟讀專注感。
- 計數器成就標籤：每句練習次數改為圓形徽章樣式。
- 橫向模式優化：手機 Landscape 會壓縮控制塔高度，保留更大影片區並維持至少一行字幕可見。
- 可讀性優化：字幕/按鈕/說明全面使用 `system-ui`，最小字級 16px，行高 1.6。
- 置中校準：依控制塔實際高度動態計算可視中心與 `scroll-margin-top`。
- 盲聽修復：模糊效果僅作用於字幕文字，不影響按鈕與卡片框。
- 桌面佈局重構：改為 `h-screen` 上下 Flex 分區（上固定區 / 下字幕獨立捲動區），從根源解決遮擋。
- 寬螢幕閱讀優化：字幕區限制為 `max-w-4xl` 等級寬度，提升長句可讀性。
- 捲動條主題化：套用深色像素風自訂 scrollbar，與整體風格一致。
- 結構語義化：主容器採 `#app-container`（`h-screen flex flex-col`），上方 `flex-none`、下方 `flex-1 overflow-y-auto`。
- 電腦端高度管控：上方影片/控制區在 Desktop 限制為 `max-height: 50vh`，避免壓縮字幕閱讀區。
- 單句控制回歸：語速、循環、播放/暫停事件重新校準，確保在新容器中可穩定觸發。
- 操作群組化：上一句/重播/下一句與全局語言/盲聽開關分組呈現，辨識更清楚。
- 全球語言支援：雙欄字幕改為語言中性（L1/L2）流程，適用任意語言配對。
- 隱藏播放器模式：可一鍵收起播放器，保留控制與字幕，適合盲聽反覆訓練。
- 練習歷史管理：標題化歷史圖書館（桌機 hover 彈出、手機 bottom sheet）可快速重載或刪除紀錄項目。
- 計數管理：提供「重設計數」僅清空當前影片句子計數，不影響字幕內容。
- 防誤觸保護：所有清洗/刪除操作加入二次確認彈窗。
- 自動續練：重新整理會自動載入上次影片，並回到上次播放秒數（或當前句起點）。
- 左右欄位獨立捲動：`#left-panel` 與 `#subtitle-container` 分離 `overflow-y-auto`，桌機/手機都可穩定滾動不互相卡死。
- 意圖導向交互：卡片 hover 只做輕微預覽，點擊才觸發解模糊、跳句與置中。
- 事件委派架構：字幕按鈕由父容器單一 listener 處理，重繪後仍可用、無重複綁定。
- 事件委派強化：字幕按鈕監聽提升到 `document.body`，並加入 `data-id` 防呆與 debug log。
- 單一狀態來源：字幕、目前播放索引、循環目標、循環次數、主語速統一收斂於 `state` 物件。
- 字幕識別改為 `data-id`：渲染與事件流使用穩定 `id`，不再依賴可漂移的 `index`。
- 鍵盤快捷鍵：`Space` 播放/暫停、`←` 上一句、`→` 下一句、`R` 切換當前句循環。
- 左側分層控制：Playback / Display / System 三區塊，操作路徑更清楚。
- 主題切換（`🎨 風格`）：可循環切換 `暖系文青`（預設）/ `硬派像素` / `商業專業`，並記憶到 `localStorage`。
- 單句功能鍵穩定化：改為父層事件委派，確保切片重渲染、切影片、切主題後，語速/循環/播放/語言切換仍可用。
- 圓角階層規範：一般按鈕/輸入 8px、卡片/面板 12px、Badge 才可全圓角，移除泡泡化外觀。
- 左側隱藏修復：改為 `display: none` 隱藏左側，右側字幕區 `flex: 1` + `width: 100%` 自動延展，並提供浮動「顯示左側」按鈕。

## 檔案結構

- `README.md`：專案說明與維護紀錄。

## 如何在本機使用

1. 直接用瀏覽器開啟 `index.html`。
2. 貼上 YouTube 網址（例如 `https://www.youtube.com/watch?v=...`）。
3. 按「載入影片與字幕」。
4. 看到字幕列表後：
   - 點「跳到這句」可跳轉播放。
   - 點 `🔁 循環` 可重播該句，點「停止循環」可解除。
   - 開啟 Auto-Pause 可自動句尾留白並接續下一句。
   - 使用 `上一句 / 重播 / 下一句` 快速操控。
   - 每句右側會顯示累積練習次數。
5. 若自動抓字幕失敗：
   - 點「手動貼上字幕」。
   - 貼上 `00:12 句子` 或每行純文字。
   - 按「套用手動字幕」即可練習。

## 如何免費部署到 GitHub Pages（手機可直接連）

1. 在 GitHub 建立新 repository（例如 `youtube-shadowing-player`）。
2. 把目前專案檔案上傳（至少 `index.html`、`README.md`）。
3. 到 repository 的 **Settings** → **Pages**。
4. **Build and deployment** 選：
   - Source: `Deploy from a branch`
   - Branch: `main`（或你的分支）/ root
5. 儲存後等待 1~3 分鐘，會出現網址：
   - `https://<你的帳號>.github.io/<repo名稱>/`
6. 用手機開啟該網址即可使用。

## 注意事項（重要）

- 由於字幕 API 是第三方服務，可能受限於：
  - CORS 規則
  - 配額限制
  - 回傳格式變動
- 如果某天無法抓字幕，不代表網頁壞掉，可能是 API 服務策略改變。可替換 `index.html` 內 `providers` 陣列中的來源。

## UX 架構（重構版）

- **事件委派**：`#subtitle-container` 僅綁定一次 click listener，透過 `data-action` + `data-index` 派發動作。
- **卡片精簡**：單句卡片只保留 `jump / play / loop`，避免過多微操作按鈕造成誤觸。
- **控制分層**：
  - Playback：上一句、播放/暫停、重播、下一句、Master Speed、Auto Pause
  - Display：L1/L2 顯示、盲聽、聚焦、隱藏播放器、主題切換
  - System：清空本片進度、清除歷史
- **快捷鍵**：輸入框聚焦時不觸發；其餘情況支援空白鍵與方向鍵快速操控。
